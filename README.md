## XamainAndroidBindLibary
自己做的一些安卓绑定好的第三方库并带demo的项目
希望能够帮助大家。<br>
使用demo进行更改首先考虑 applicaiton 然后是AndroidManifest.xml<br>
里面的很多细节请注意复制和查看尤其是包名
以下demo都是以真机为主。如果想在模拟器运行。请注意拷贝x86相关so文件到lib中
